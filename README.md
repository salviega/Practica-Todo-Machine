## Empezar 🏄

Prerequisitos: [Node (v16 LTS)](https://nodejs.org/en/download/) y [Git](https://git-scm.com/downloads)

> Clona el repositorio:

```
git clone https://github.com/salviega/Practica-Todo-Machine.git
```

> Instala las dependencias:

```
npm install
```

> Corre el proyecto:

```
npm start
```
